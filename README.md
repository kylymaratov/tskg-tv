# TS.KG Android TV

A simple app for browsing and watching film series and anime (ad‑free).

## ✅ What it is
- Android TV focused UI (Leanback-style navigation)
- Local data source (`assets/movies.json`) for movies/series
- Built with Kotlin and Android Jetpack components

## 🚀 Getting started
1. Open the project in Android Studio
2. Build and run on an Android TV device or emulator

## 🗂️ Project structure
- `app/src/main/java` – app code (activities, adapters, models, parser)
- `app/src/main/res` – layouts, drawables, navigation, and resources
- `app/src/main/assets/movies.json` – sample content data

## ℹ️ Notes
- Not an official app; intended for personal use.
- Future migration planned to React Native.
